# To-Do List App – Dokumentation

## Installation
Keine Abhängigkeiten. Einfach `python todo_app.py` ausführen.

## Nutzung
- `add` – Neue Aufgabe eingeben.
- `list` – Alle Aufgaben nummeriert anzeigen.
- `done` – Aufgabe nach Nummer als erledigt markieren (wird entfernt).
- `quit` – Programm beenden.

## Beispiel
```
$ python todo_app.py
Command (add/list/done/quit): add Milch kaufen
Command (add/list/done/quit): add Code schreiben
Command (add/list/done/quit): list
1. Milch kaufen
2. Code schreiben
Command (add/list/done/quit): done 1
Done: Milch kaufen
Command (add/list/done/quit): quit
```
