# 🔑 API-Key-Börse — Produktidee

**Datum:** 14. Mai 2026  
**Status:** Idee  

---

## Was

Plattform zum Kaufen, Verkaufen und Verwalten von KI-API-Keys. Ein Marktplatz für AI-Zugang.

## Warum

Jeder KI-Nutzer hat 5+ Keys über verschiedene Anbieter verstreut. Kein Überblick, keine Deals, keine Flexibilität.

## Features

- **One-Stop-Shop:** DeepSeek, OpenAI, Anthropic, ElevenLabs, Mistral, Google — alles an einem Ort
- **Deals:** Volumen-Rabatte durch Sammelbestellungen bei Anbietern
- **Transfer:** Key an andere User umschreiben / weitergeben
- **Rückgabe:** Ungenutztes Guthaben zurückgeben
- **Aktionen:** Flash-Sales, Bundles, Free-Tier-Starter-Pakete
- **Dashboard:** Verbrauch aller Keys auf einen Blick, Kosten-Alarm

## Geschäftsmodell

- Provision pro Transaktion (2-5%)
- Premium-Dashboard (Abo)
- Volumen-Marge (Einkauf günstiger als Verkauf)
