# Gnom-Hub Arbeitsbereich

Dieses Verzeichnis ist der Arbeitsbereich für Gnom-Hub.
Setze beim Start den Arbeitsordner auf /Users/landjunge/gnom-hub.
Der Server läuft auf Port 3002.

## WICHTIG: Vor jedem git push

Die Datei `PRE_PUSH_CHECKLIST.md` lesen und ALLE Schritte abarbeiten:
1. Hub stoppen + Monitor killen
2. DB leeren (unnötige Laufzeitdaten löschen)
3. Backup-Dateien + Caches entfernen
4. Keine Secrets im Code prüfen
5. README-Badges aktualisieren
6. Tests laufen lassen (154, alle pass)
7. Install/Uninstall testen
8. Git clean committen
9. Letzten Check vor Push
10. Push
